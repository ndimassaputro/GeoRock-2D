# Literature Matrix

This matrix records selected references supporting the scientific motivation,
inverse-problem formulation, rockhead interpretation, and borehole–geophysics
integration used in GeoRock-2D.

## Reference Integrity Statement

- DOI links were checked against publisher or professional conference pages.
- Model dimensions, velocity values, noise levels, velocity thresholds, and
  borehole influence lengths in GeoRock-2D are synthetic assumptions.
- GeoRock-2D does not implement the full methods described in the cited
  Bayesian, geological-modelling, or seismic-tomography studies.
- The references provide scientific context rather than direct validation of
  the present conceptual V-shaped ray implementation.
